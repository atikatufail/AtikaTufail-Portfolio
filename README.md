## ✨ Features

- **🎯 Smart Project Showcase**: Categorized projects (Full-stack, UI/UX, Mobile) with dynamic button rendering.
- **📄 Dual Resume Viewer**: Real-time PDF preview and download options for both .NET and Full Stack specialized resumes.
- **🛠️ Interactive Tech Stack**: Visually stunning "Icon Cards" representing expertise in AI, Data Science, and Web Development.
- **🎨 Custom Branding**: Personal "AT" logo branding with a cohesive premium pink and black aesthetic.
- **📱 Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.
- **⚡ High Performance**: Built with **Vite** for near-instant load times and optimized asset delivery.

---

## 🛠️ Technology Stack

- **Core**: [React.js](https://reactjs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [React-Bootstrap](https://react-bootstrap.github.io/) & Vanilla CSS3
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [Particle.js](https://vincentgarreau.com/particles.js/), & [Typewriter-effect](https://www.npmjs.com/package/typewriter-effect)
- **Icons**: [React-Icons](https://react-icons.github.io/react-icons/) (FontAwesome, Ant Design, Material Design)
- **PDF Core**: [React-PDF](https://projects.wojtekmaj.pl/react-pdf/)

## ⚙️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/atikatufail/myPortfolio.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd myPortfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run in development mode**:
   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```
